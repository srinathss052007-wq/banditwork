A cron job runs a script automatically as bandit23. The script reads the password and writes it to a predictable file in /tmp. Reading that file reveals the password. commands used : cat /etc/cron.d/cronjob_bandit23 cat /usr/bin/cronjob_bandit23.sh cat /tmp/<generated_filename>
