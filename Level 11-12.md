Level 11-12 :

The password for the next level is stored in the file data.txt, which contains base64 encoded data. Base64 encoding is a method of encoding binary data, such as files or non-text data, into a text-based format. 

![level 10-level 11](https://github.com/user-attachments/assets/d14ddee6-b0b0-4baf-a2a2-98a43c48e809)

To accommodate URL parameters and avoid potential encoding conflicts, URL-safe Base64 replaces '+' with '-' and '/' with '_'.
